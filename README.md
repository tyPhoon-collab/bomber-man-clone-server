# Bomber Man Clone

Server:
Golang, Socket.IO

Client:
TypeScript, Socket.IO-Client, Three.js

## Env

```bash
$ go version
go version go1.22.5 darwin/arm64

$ node --version
v20.10.0
```

## Setup

```bash
npm install
go mod tidy
```

Two shells:

- `cd client && npx vite`
- `go run .`

After that, access at `http://localhost:5137`
